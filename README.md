# Graze & Grain Restaurant App

## 🚀 Quick Start

### Backend API (localhost:3001)

```bash
cd backend
npm install
node index.js
```

### Frontend (localhost:3000)

```bash
cd frontend
npm install
npm run dev
```

## ✨ Features

- 🍽️ Full menu with cart/checkout
- 👨‍💼 Admin panel (login, CRUD menu/orders/reservations)
- 🛒 Persistent cart (localStorage)
- 📱 Responsive design (TailwindCSS + shadcn/ui)
- 💾 JSON database
- 🖼️ Local images (no external 404s)
